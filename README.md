## Instruction to Run the Project Locally
1. First Clone the repository
```
git clone https://github.com/FarhanAsfar/W3E-Assignment3.git
```

2. Navigate to the project directory

3. Open the project folder in VS CODE (or your preferred code editor)

4. If you have 'Live Server' extension installed in your editor, then you can just clikc on the 'Go Live' icon at the bottom right of vs code.
OR,
Open the terminal on vs code, then run 'pwd'. You will get the path of the project. Now, copy the path and paste it in your browser (e.g: /home/downloads/W3E-Assignment3/index.html) and you will see the project live.
